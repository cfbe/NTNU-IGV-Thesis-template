# Thesis template for IGV, NTNU #

This is a template for students at IGV, NTNU, to use when writing their specialization report and master thesis. The template is also appropriate for students in other departments.

This template is also found on overleaf:
[https://www.overleaf.com/latex/templates/ntnu-igv-thesis-template/sndrhyxmsnvj](https://www.overleaf.com/latex/templates/ntnu-igv-thesis-template/sndrhyxmsnvj)

This is a template, so please use it as such: you can change and amend it the way you like.

This is based on an earlier template by Nina Salvesen (https://www.overleaf.com/latex/templates/thesis-template-ntnu/kpybjxchxnmc).
